Google Drive — Auto Upload

One-time setup (5 min):
1. Go to drive.google.com
2. Click New → Folder → name it "Kamryn & Ted Wedding Photos"
3. Right-click the folder → Share → Change to Anyone with the link → Viewer
4. Click Copy link — paste it below
5. The Folder ID is the long string in that URL after /folders/
In Google Cloud Console:
• APIs & Services → Library → enable Google Drive API
• Credentials → OAuth Client ID → Web application
• Add your GitHub Pages URL as Authorized JavaScript Origin
• In Data Access, add scope: drive.file
(This scope does NOT require Google verification ✓)
